# 🧬 Traducteur ADN → Protéine

Un outil Python puissant pour convertir des séquences d'ADN en séquences de protéines selon le code génétique standard.

## ✨ Fonctionnalités

- ✅ Traduction ADN → Protéine selon le code génétique standard
- 🔍 Recherche automatique du codon start (ATG)
- 🛑 Arrêt aux codons stop (TAA, TAG, TGA)
- 📊 Analyse des 3 cadres de lecture possibles
- 📈 Statistiques détaillées de la protéine (composition, longueur)
- 🗂️ Lecture/écriture de fichiers (format FASTA et texte)
- ⚠️ Gestion complète des erreurs
- 🎯 Validation des séquences ADN
- 📝 Interface simple et intuitive
